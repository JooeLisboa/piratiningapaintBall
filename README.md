# Arena Paintball - Piratininga PaintBall

Site institucional/comercial em Next.js com foco em conversão via WhatsApp.

## Rodando localmente

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`.

## Produção

```bash
npm run build
npm run start
```

## Onde editar conteúdo

- Dados centralizados em `data/siteContent.ts`
- Mensagem/URL de WhatsApp em `lib/whatsapp.ts`
- Seções em `components/sections/*`
